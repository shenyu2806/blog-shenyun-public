|浏览量:

前言

本站有使用API图片接口，推荐的接口与本站无关！

Jitsuの涩图接口[随机涩图]

地址：Jitsuの随机涩图

图片数量统计：
全部（all）：20000+
横屏壁纸（pc）：1000+
竖屏壁纸（mp）：2000+
银发（silver）：1500+
兽耳（furry）：2000+
星空（starry）：500+
涩图（setu）：4000+
1080P（1080）：3000+
P站随机图（pixiv）：50000+
18+（r18）：10000+
我的收藏（jitsu）：6000+

调用示例

  • 直接调用 https://moe.jitsu.top/img
  • 类别数量 https://moe.jitsu.top/img/?sort=pixiv&num=3
  • 设置规格 https://moe.jitsu.top/img/?sort=pc&size=mw1024
  • 返回json https://moe.jitsu.top/img/?sort=r18&size=1080p&type=json
  • 设置反代 https://moe.jitsu.top/img/?sort=jitsu&size=original&type=json&num=5&proxy=i.pixiv.re

樱道随机图片API接口

地址：樱道随机图片API

调用示例：

  • 动漫综合: https://api.r10086.com/樱道随机图片api接口.php?图片系列=动漫综合1
  • 赛马娘: https://api.r10086.com/樱道随机图片api接口.php?自适应图片系列=赛马娘
  • 我的世界系列: https://api.r10086.com/樱道随机图片api接口.php?图片系列=我的世界系列1
